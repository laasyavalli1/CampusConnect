Features

Skill Sharing Module

  • Branch/topic-based chat groups for discussions.
  • Resource sharing (notes, references, tutorials).
  • Mentorship system for juniors to connect with seniors.
  • User profiles to showcase interests and expertise.

Campus Navigation Module

  • Interactive map with zoom and search functionality.
  • Shortest path finding between two campus locations.
  • Category-based search (hostels, departments, canteens).
  • Nearby facilities listing (ATM, medical center, etc.).
